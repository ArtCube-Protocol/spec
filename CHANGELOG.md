# Changelog

All notable changes to the ArtCube Protocol specification will be documented in this file.

## [1.0] - 2026-02-25

### Added
- Initial protocol specification (whitepaper v1.0)
- Four-level inscription hierarchy: Entity Root, Collection Parent, Genesis Title Anchor, Events
- Eight event types: Title, Document, Condition, Custody, IP, Metadata Update, Correction, Supersession
- Hybrid title transfer model (UTXO + Title Event)
- Intellectual property framework with baseline display rights
- Three-tier document storage layer (Bitcoin + Arweave + off-chain)
- Verification algorithm pseudocode
- JSON schemas for all inscription types
- Example inscriptions (Battle of the Centaurs)
